弹出层api参数说明

弹出层

layer.alert(Object)

一个模拟浏览器自带弹出效果的弹出层，可选中title区域进行移动。（所有有默认值的参数，都是可选参数）

layer.alert({type : 1, title : '第一个弹出层', content : '我们来写点什么'});

type

类型 : Number

设置弹出层类型。值为1-3整数数字，1表示提示(alert)、2表示询问(confirm)、3表示可输入交互(prompt)

title

类型：String

需要显示的标题信息。默认值：信息

content

类型 : String

需要显示的内容信息。默认值：内容展示

width

类型 : Number

弹出框自定义宽度。默认值：260

height

类型 : Number

弹出框自定义高度。默认值：auto

conType

类型 : String

弹出框类型为3时的input框类型。默认值：text

link

类型 : String

按钮连接地址，对确定按钮有效。默认值：javascript:void(0);

target

类型 : String

按钮打开方式，对确定按钮有效。默认值：_self

btn

类型 : Array

按钮文字。默认值：['确定','取消']

right

类型 : Function

确定按钮回调函数，不返回false时触发关闭事件。默认值：function(){}

cancel

类型 : Function

取消按钮回调函数，不书写时或类型不是function时，不显示此按钮。默认值：null

layer.tips(Object)

一个简单的气泡、提示组件，可按住alt阻止自动消失。会返回当前对象。（所有有默认值的参数，都是可选参数）

var tips = layer.tips({target : $(ele), content : '第一个tips'});

content

类型 : String

需要显示的内容信息。默认值：hi, 这是一个小的tips

direction

类型：String

显示方位，支持：left,top,right,bottom,center。默认值：content

auto

类型 : Boolean

是否自动消失。默认值：true

more

类型 : Boolean

是否运行多个气泡显示。默认值：false

delay

类型 : Number

自动消失时间。默认值：1000

hide(delay)

类型 : Function

返回的对象方法，用于操作影藏。可传入延时时间

var tips = layer.tips();

layer.loading(Object)

一个只适应的loading层，会返回当前对象。（所有有默认值的参数，都是可选参数）

content

类型 : String

需要显示的内容信息。默认值：hi, 这是一个小的tips

hide()

类型 : Function

返回的对象方法，用于操作影藏。

var load = layer.loading();
load.hide();//用于影藏当前loading层
